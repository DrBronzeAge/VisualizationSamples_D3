These are the sourcecode for a couple of D3 visualizations that I did for a colleague. Running versions can be found here:

http://jineel1.wix.com/libraryofantiquity
The text of the analysis that they support can be found here:
https://libraryofantiquity.wordpress.com/2016/03/18/trends-in-classics-a-sparagmos/

The data set on which they are founded is a combination of full-text reviews and catalogue entries for
 ~15000 bookspublished in the field of classics between 1995 and 2016, which I scraped and parsed.
